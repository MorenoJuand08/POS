# SISTEMA_POS 
( Sistema de Punto de Venta )

Trendo POS es software para la gestión de inventario y ventas en tiendas de moda. Permite controlar stock por tallas, registrar ventas en caja, generar reportes contables y trabajar incluso sin conexión, sincronizando los datos con la nube cuando vuelve el internet.

Tecnologías utilizadas
JavaScript (ES2021+) para todo el desarrollo de la aplicación.
Electron para empaquetar y ejecutar el POS como app de escritorio en Windows.
React para la interfaz de usuario.
TailwindCSS para los estilos y el diseño responsive.
SQL (PostgreSQL a través de Supabase) para la base de datos en la nube.
